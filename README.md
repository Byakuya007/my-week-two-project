# AkAN NAMES WEBSITE

## AUTHOR
Dennis Collins Thumbi

### PROJECT DESCRIPTION 
This is a website that is is based on the Ghanian Akan naming system. the web interface asks the user to input his/her birthday,select the gender then proceed to submit the data.

### TECHNOLOGIES USED
1.HTML
2.CSS
3.GIT 
4.BOOTSTRAP
5.JAVASCRIPT
